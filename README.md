# ChatGPT Example

## 1. 项目初始化
- [项目工程的初始化过程](./INITIAL.md)

## 2. 安装依赖
```bash
pnpm install
```

## 3. 启动开发
```bash
pnpm dev
```

## 4. 打包产物
```bash
pnpm build
```
